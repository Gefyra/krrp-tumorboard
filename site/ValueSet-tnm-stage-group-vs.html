ValueSet: TNM Stage Group Value Set

Official URL: http://krebsregister-rlp.de/fhir/tumorboard/ValueSet/tnm-stage-group-vs Version: 0.1.0
Draft as of 2024-02-29 Computable Name: TNMStageGroupVS

Result values for cancer stage group using TNM staging. This value set contains SNOMED-CT equivalents of UICC codes for Stage Group, according to TNM staging rules.

References

  • Included into TNMStageGroupMaxVS
  • TNM Stage Group Profile

Logical Definition (CLD)

This value set includes codes based on the following rules:

  • Include codes from http://snomed.info/sct where concept descends from 1222592004 (American Joint Committee on Cancer clinical stage group allowable value)
  • Include codes from http://snomed.info/sct where concept descends from 1222593009 (American Joint Committee on Cancer pathological stage group allowable value (qualifier value))
  • Include codes from http://snomed.info/sct where concept descends from 1222594003 (American Joint Committee on Cancer yp stage group allowable value (qualifier value))
