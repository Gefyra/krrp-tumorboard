Result values for T category. This value set contains SNOMED-CT equivalents of UICC codes for the T category, according to TNM staging rules.

Logical Definition (CLD)

This value set includes codes based on the following rules:

  • Include codes from http://snomed.info/sct where concept descends from 1222585009 (American Joint Committee on Cancer clinical T category allowable value (qualifier value))
  • Include codes from http://snomed.info/sct where concept descends from 1222589003 (American Joint Committee on Cancer pathological T category allowable value (qualifier value))
  • Include codes from http://snomed.info/sct where concept descends from 1222595002 (American Joint Committee on Cancer ypT category allowable value (qualifier value))
